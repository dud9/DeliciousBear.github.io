个人博客的搭建(Hexo)

Step one. 安装

1.首先安装hexo cnpm install -g hexo-cli (node.js)

2.mkdir blog 创建文件夹

3.执行hexo init

4.启动博客 hexo s (hexo start)

Step two. 新建文章

使用 hexo new “xxx” 命令

写完文章后

hexo clean

hexo g –生成
